LG U+ MVNO Business Automation

LG U+ 사무 인턴십 기간 중 수행한 MVNO(알뜰폰) 사업팀 업무 자동화 프로젝트. 웹 크롤링, 감성 분석, 자동 리포트 시스템 구축.

2025 LG U+ Internship Business Automation

Overview

LG U+ MVNO 사업팀에서 사무 인턴으로 근무하며 경쟁사 요금제 모니터링, 뉴스 감성 분석, 커뮤니티 여론 분석 업무를 자동화했습니다. 기존 수작업으로 수 시간이 소요되던 반복 업무를 Python 기반 자동화 시스템으로 대체하여 업무 효율성을 대폭 개선했습니다.

Business Impact

매일 아침 수동으로 진행하던 경쟁사 요금제 조사를 자동화하여 GitHub Actions 기반 매일 자동 실행 및 이메일 리포트 발송 시스템 구축. 업무 시간 절감 및 데이터 정확성 향상.

Key Deliverables

1

MVNO Promotion Crawler

7개 경쟁사 사이트(모요플랜, 아이즈, 프리티, 모빙, 스마텔, 이야기, 티플러스)에서 요금제 정보를 자동 수집하고 비교 분석 리포트 생성.

3

News Sentiment Analysis

Naver API로 알뜰폰 관련 뉴스 수집, TF-IDF 중복 필터링, HuggingFace 감성 분석 모델로 긍정 뉴스 자동 추출.

4

Community Sentiment Monitoring

뽐뿌 커뮤니티에서 MVNO 관련 게시글 수집, KOTE 감성 분석 모델로 44개 감정 라벨 분류 및 여론 분석.

Data Processing Pipeline

MoyoplanScraper - Core Logic
class MoyoplanScraper:
    def __init__(self, search_terms=None):
        self.search_terms = search_terms or DEFAULT_SEARCH_TERMS
        self.driver = setup_chrome_driver()

    def scrape_search_term(self, search_term: str):
        """검색어별 요금제 데이터 수집"""
        url = f'{BASE_URL}?searchText={search_term}&sorting=fee_asc'
        self.driver.get(url)
        scroll_to_bottom(self.driver)

        # 페이지네이션 처리
        for page in range(1, total_pages + 1):
            soup = BeautifulSoup(self.driver.page_source, 'html.parser')
            data = extract_plan_data(soup, search_term)
            self.scraped_data.extend(data)

        return self.scraped_data

    def process_dataframe(self, df):
        """데이터 정제 및 피벗 테이블 생성"""
        df['데이터크기_범주'] = df.apply(categorize_data_size, axis=1)
        pivot_df = df.pivot_table(
            index='통신사이름',
            columns='데이터크기_범주',
            values=['프로모션가격', '할인개월수', '원가']
        )
        return pivot_df

GitHub Actions Automation

크롤링 작업을 GitHub Actions로 자동화하여 매일 정해진 시간에 실행되도록 구성했습니다. 수집된 데이터는 Excel 파일로 생성되어 담당자에게 이메일로 자동 발송됩니다.

.github/workflows/daily-crawl.yml
name: Run Moyoplan Daily

on:
  schedule:
    - cron: '30 22 * * *'  # UTC 22:30 = KST 07:30
  workflow_dispatch:        # 수동 실행 지원

jobs:
  run-script:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3

      - name: Set up Python
        uses: actions/setup-python@v4
        with:
          python-version: '3.10'

      - name: Install dependencies
        run: pip install -r requirements.txt

      - name: Run crawler
        env:
          SENDER_EMAIL: ${{ secrets.SENDER_EMAIL }}
          EMAIL_PASSWORD: ${{ secrets.EMAIL_PASSWORD }}
        run: python moyo_프로모션조사_구문.py

def send_email_report(excel_filename):
    msg = EmailMessage()
    msg['Subject'] = f"프로모션_{today}"
    msg['From'] = SENDER_EMAIL
    msg['To'] = RECEIVER_EMAIL
    msg.set_content(f"25년 {today} 프로모션 현황입니다.")

    # Excel 파일 첨부
    with open(excel_filename, "rb") as f:
        msg.add_attachment(f.read(),
            maintype="application",
            subtype="vnd.openxmlformats-officedocument.spreadsheetml.sheet",
            filename=excel_filename)

    # Gmail SMTP로 발송
    with smtplib.SMTP("smtp.gmail.com", 587) as conn:
        conn.starttls()
        conn.login(SENDER_EMAIL, EMAIL_PASSWORD)
        conn.send_message(msg)
