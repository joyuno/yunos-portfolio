MovieMania

Spring Boot 기반 영화 커뮤니티 웹 애플리케이션. 자유게시판, 영화 리뷰 게시판, 시사회 신청/추첨 기능을 구현한 풀스택 프로젝트.

2024 Team Project Full Stack

Overview

MovieMania는 영화 팬들을 위한 커뮤니티 플랫폼입니다. 사용자들이 자유롭게 소통하고, 영화 리뷰를 작성하며, 시사회에 응모할 수 있는 기능을 제공합니다. Spring Boot를 활용한 백엔드와 JSP 기반 프론트엔드로 구성된 풀스택 웹 애플리케이션입니다.

Project Goals

Spring Framework 생태계에 대한 이해와 실무 수준의 웹 애플리케이션 개발 경험 축적. 사용자 인증, 권한 관리, CRUD 게시판, 추첨 시스템 등 다양한 기능 구현.

Key Features

1

Multi-Board System

자유게시판, 영화 리뷰 게시판, 공지사항 게시판 등 다양한 목적의 게시판 구현. 페이지네이션, 검색 기능 포함.

2

Premiere Application System

시사회 정보 등록 및 사용자 응모 기능. 관리자 추첨 시스템으로 당첨자 자동 선정.

3

User Authentication

Spring Security 기반 로그인/회원가입. 역할(ROLE) 기반 권한 관리로 일반 사용자와 관리자 구분.

4

Social Features

댓글, 좋아요, 쪽지 기능으로 사용자 간 상호작용 지원.

System Architecture

전통적인 MVC 패턴을 기반으로 한 3계층 아키텍처로 설계되었습니다. Controller, Service, Repository 레이어로 관심사를 분리하여 유지보수성을 높였습니다.

Implementation Details

Spring Security Configuration

사용자 인증 및 권한 관리를 위해 Spring Security를 적용했습니다. 폼 기반 로그인과 역할(ROLE) 기반 접근 제어를 구현했습니다.

SecurityConfig.java
@Configuration
@EnableWebSecurity
public class SecurityConfig extends WebSecurityConfigurerAdapter {

    @Override
    protected void configure(HttpSecurity http) throws Exception {
        http
            .authorizeRequests()
                .antMatchers("/admin/**").hasRole("ADMIN")
                .antMatchers("/board/**").hasAnyRole("USER", "ADMIN")
                .antMatchers("/", "/login", "/register").permitAll()
            .and()
            .formLogin()
                .loginPage("/login")
                .defaultSuccessUrl("/")
            .and()
            .logout()
                .logoutSuccessUrl("/");
    }
}

Premiere Lottery System

시사회 응모자 중 당첨자를 무작위로 선정하는 추첨 시스템을 구현했습니다. 공정한 추첨을 위해 SecureRandom을 활용했습니다.

PremiereService.java
public List<PremiereApplicationDTO> drawWinners(int premiereId, int count) {
    List<PremiereApplicationDTO> applicants =
        applicationMapper.getApplicantsByPremiereId(premiereId);

    Collections.shuffle(applicants, new SecureRandom());

    List<PremiereApplicationDTO> winners =
        applicants.stream().limit(count).collect(Collectors.toList());

    winners.forEach(winner -> {
        winner.setStatus("WINNER");
        applicationMapper.updateStatus(winner);
    });

    return winners;
}

MyBatis Mapper

데이터베이스 접근을 위해 MyBatis를 사용했습니다. XML 기반 매퍼로 복잡한 쿼리를 관리하고, 동적 SQL을 활용했습니다.

BoardMapper.xml
<select id="searchBoards" resultType="BoardDTO">
    SELECT * FROM freeboard
    WHERE 1=1
    <if test="searchType == 'title'">
        AND title LIKE CONCAT('%', #{keyword}, '%')
    </if>
    <if test="searchType == 'content'">
        AND content LIKE CONCAT('%', #{keyword}, '%')
    </if>
    <if test="searchType == 'writer'">
        AND writer LIKE CONCAT('%', #{keyword}, '%')
    </if>
    ORDER BY num DESC
    LIMIT #{offset}, #{limit}
</select>
