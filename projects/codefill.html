CodeFill - AI 코딩 학습 플랫폼

LangGraph 기반 Multi-Agent 오케스트레이션 시스템을 활용한 개인화 코딩 학습 플랫폼. RAG, Collaborative Filtering, Progressive Hint System을 통합한 풀스택 AI 서비스.

2025 Personal Project Full Stack AI

Overview

CodeFill은 LLM(Large Language Model)과 Multi-Agent 아키텍처를 활용하여 사용자 맞춤형 코딩 문제를 제공하고 학습 과정을 지원하는 AI 기반 교육 플랫폼입니다. LangGraph StateGraph를 활용한 복잡한 대화 흐름 관리와 RAG 기반 문제 검색, 협업 필터링 추천 시스템을 통합한 풀스택 서비스입니다.

Project Goals

교육 도메인에서의 AI 적용 - 소크라테스식 교수법을 적용한 점진적 힌트 시스템, 사용자 학습 이력 기반 개인화 추천, Multi-Agent 협업을 통한 학습 지원 자동화 구현.

Core Features

1

LangGraph Multi-Agent Orchestration

4-Stage Pipeline(Intent→Collection→Discovery→Solving)을 통한 복잡한 대화 흐름 관리. Conditional Edges 기반 동적 라우팅과 PostgreSQL 체크포인트 영속성 구현.

2

RAG Pipeline with Hybrid Search

pgvector 기반 Vector Search와 메타데이터 필터링의 Hybrid Search. Similarity threshold(0.30) 이하 시 LLM CodeGen Fallback 전략 구현.

3

Collaborative Filtering Recommendation

Cosine Similarity(주제 선호도) + Jaccard Similarity(주제 겹침) + Temporal Decay(시간 가중치) 기반 유사 사용자 추천. A/B Testing 통합.

4

Progressive Hint System

소크라테스식 교수법을 적용한 4단계 점진적 힌트(10%→40%→70%→90% 공개). 자기 주도적 문제 해결 촉진.

System Architecture

LangGraph Orchestrator V2를 중심으로 한 계층형 아키텍처로 설계되었습니다. 각 Agent는 독립적인 역할을 수행하며, StateGraph를 통해 유기적으로 협업합니다.

Multi-Agent System

Agent Model Role
Intent Tool gpt-4o-mini 사용자 의도 분류 (category + action)

Implementation Details

LangGraph StateGraph Pipeline

4-Stage Pipeline을 통해 사용자 입력을 처리합니다. 각 단계는 독립적인 Graph로 구현되어 Conditional Edge를 통해 동적으로 라우팅됩니다.

orchestrator_v2.py
# 4-Stage Pipeline Architecture
Orchestrator V2
├── Intent Classification (의도 분류)
│   └── category: discovery | solving | general
├── InfoCollectionGraph (정보 수집)
│   └── topic, difficulty, language 추출
├── DiscoveryGraph (문제 탐색)
│   └── RAG Search → CodeGen Fallback
└── SolvingGraph (문제 풀이 지원)
    └── Hint | Check | Feedback

RAG Hybrid Search Pipeline

Vector Similarity와 메타데이터 필터링을 조합한 Hybrid Search로 관련 문제를 검색합니다. 유사도 임계값(0.30) 미만 시 CodeGen Agent가 새 문제를 생성합니다.

rag.py
Query → Embedding → Filter (topic/difficulty) → Vector Search → Rank → Results
                                                      ↓
                                            (similarity < 0.30)
                                                      ↓
                                              CodeGen Fallback

# Embedding Service - 코드 구조 추출 및 청킹
class EmbeddingService:
    async def generate_embedding(self, text: str) -> List[float]
    def create_problem_text_for_embedding(self, problem: Dict) -> str
    def chunk_code(self, code: str, chunk_size: int = 500) -> List[str]
    def _extract_code_structure(self, code: str, language: str) -> str

Collaborative Filtering

유사 사용자 기반 추천 시스템으로 개인화된 문제 추천을 제공합니다. A/B Testing을 통해 알고리즘 성능을 비교합니다.

collaborative_filtering.py
# 유사도 계산 알고리즘
Final Score = (
    Cosine Similarity (topic preferences) × 0.40 +
    Jaccard Similarity (topic overlap)    × 0.20 +
    Metadata Matching (goal/level)        × 0.30 +
    Activity Weight (interaction count)   × 0.10
)

# Temporal Decay: 최근 상호작용에 높은 가중치 (지수 감쇠)
# Diversity Sampling: Top-N에서 가중치 랜덤 샘플링

Progressive Hint System

소크라테스식 질문 유도를 통해 자기 주도적 문제 해결을 촉진합니다. 단계별로 점진적으로 정보를 공개하여 학습 효과를 극대화합니다.

hint_agent.py
Level 1: 방향 힌트     (10-20% 공개) - "어떤 자료구조를 생각해보세요"
Level 2: 접근법 힌트   (30-40% 공개) - "해시맵을 활용해보세요"
Level 3: 구체적 힌트   (50-70% 공개) - 의사코드 수준 설명
Level 4: 최종 힌트     (80-90% 공개) - 핵심 코드 일부 제공
